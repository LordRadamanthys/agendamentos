# Agendamentos

Esse app foi feito com base no estabelecimento da minha mãe que é cabeleireira para auxiliar ela no gerenciamento dos clientes com reservas e etc.

# Principais Tecnologias

  - **Express**: Modulo do node para montar um server de comunicação para a aplicação com a base de dados.
  - **Sequilize**: Modulo Sequilize que é um ORM para gerenciar a base de dados via JS.
  - **JsonWebToken**: Modulo para criar um token válido por 24h para ter acesso a aplicação.
  - **Firebase-admin**: Modulo para disparar Push Notification.
  - **Multer**: Modulo para gerenciar uploads de arquivos para a API.
  - **Socket**.**io**: Foi utilizado para montar um chat entre o adm do serviço e os clientes.
  - **MySql**: Base de dados.

Desenvolvimento:

> Como disse, fiz esse app para aprimorar mais os meus 
> conhecimentos no android nativo utilizando a linguagem Kotlin.
> Tinha muita expereciencia com o Java e queria ver oque essa linguagem
> tem a oferecer frente ao java. Gostei bastanate da simplicidade
> que ela propõe na hora de escrever metedos e definir classes,
> em alguns momentos tive até dificuldade por estar pensando mais 
> como Java, isso pode ter feito eu dar algumas voltas mas com 
> o tempo você vai entendendo a proposta da linguagem e suas 
> facilidades.
> Também resolvi utilizar push notifications para alertar as
> reservas e cadastros dos usuarios para os ADMs.
> E para complicar um pouco minha vida...digo... aumentar 
> meus conhecimentos resolvi fazer um chat com socket.io,
> foi bem complicado mas o resultado final ficou bom para um primeiro.
